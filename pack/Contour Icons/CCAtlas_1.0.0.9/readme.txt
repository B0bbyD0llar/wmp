   Программа позволяет разрезать Atlas на отдельные файлы, и соединить отдельный файлы в Atlas.

   Для разрезания Atlas необходимо:
1) На панели "Разделить" нажать кнопку "Обзор...";
2) В появившемся диалоге выбрать файл изображения атласа (xml файл должен находиться в этой же папке);
3) Затем нажать кнопку "Разрезать".
   После этого, в каталоге с атласом, создастся каталог с названием атласа, в котором будут находиться полученные файлы.

   Для создания атласа из отдельных файлов необходимо:
1) На панели "Соединить" нажать кнопку "Обзор...";
2) В появившемся диалоге выбрать каталог с изображениями;
3) Затем нажать кнопку "Создать".
   После этого, в выбранном каталоге, создастся папка с названием "Atlas", в котором и будут находиться файлы атласа.
Имя созданных файлов атласа, такое же, как у выбранного каталога.
